# Validação por Especialistas

## Objetivo

Este documento descreve a validação simulada por especialistas, exigida no contexto acadêmico do projeto.

## Perfis Envolvidos na Revisão Simulada

- ginecologia;
- obstetrícia;
- psicologia/perinatal;
- assistência social;
- farmácia clínica;
- governança de dados/LGPD.

## Checklist de Revisão

- terminologia médica compatível com o contexto assistencial;
- ausência de prescrição e de diagnóstico definitivo;
- encaminhamento obrigatório em urgências e violência doméstica;
- linguagem respeitosa, inclusiva e culturalmente sensível;
- proteção adequada de dados sensíveis;
- coerência entre risco, orientação e necessidade de validação profissional.

## Status no Projeto

- revisão clínica: simulada;
- revisão de segurança: implementada por regras de guardrail e testes automatizados;
- revisão de dados: simulada com anonimização e curadoria sintética;
- aprovação para uso real: não aplicável.

## Observação

O projeto não foi validado para uso assistencial real. Toda validação especializada aqui descrita serve apenas para demonstrar como o processo deveria ocorrer em ambiente hospitalar.
